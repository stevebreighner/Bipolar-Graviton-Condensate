# Bipolar Graviton Condensate

Static-site scaffold plus paper assets for the Bipolar Graviton Condensate project.

## Site layout

- `index.html`: landing page for the whole site
- `paper/`: manuscript source and compiled paper assets
- `visuals/`: full-screen conceptual visualizations
- `models/`: toy-model pages and branch sketches
- `trees/`: decision-tree apps for theory branching

## GitHub Pages

1. Open repo `Settings`.
2. Open `Pages`.
3. Set source to `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save. The site entry page is `index.html`.

## Current theory tracks

- `Option 1`: conventional-leaning framing. Do not claim matter literally becomes gravitons; treat the condensate as an effective description under extreme collapse.
- `Option 2`: transfer-field framing. Introduce a new conversion mechanism that mediates matter-to-condensate transfer, then derive its trigger conditions and conservation-law behavior.

## Tree app

- `trees/graviton-theory-decision-tree/`: decision tree for `massless vs massive/extended` and `transfer process option 1 vs option 2`.
