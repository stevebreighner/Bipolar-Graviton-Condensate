# Bipolar Graviton Condensate (BGC)

A non-singular, stable-equilibrium model for **Sagittarius A*** (Sgr A*). This repository contains the mathematical framework and numerical stability analysis for a two-component graviton fluid ($g^+/g^-$) that replaces the black hole singularity with a physical, high-density core.condensate transfer, then derive its trigger conditions and conservation-law behavior.
