Reading The Sketch

The teal and amber lobes represent opposite bipolar faces. In this "massless/original" version there is no explicit shell or hard-core boundary, so the packets can overlap more freely. With the support toggle off, the force is always inward: F(r) = -3/r^4 - 1/r^2. That means the canvas illustrates approach and collapse tendency, not a finite resting separation.

Turning the A toggle on adds the same effective support term used in the other model: +A/r^12. This is included only as a comparison, so you can see how a steep short-range contribution can create a finite equilibrium where the original massless picture does not.
