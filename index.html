Concept, Paper, and Visual Scaffolds

Bipolar Graviton Condensate

This repo now has a minimal GitHub Pages-style landing page for the paper artifacts and the HTML visualizations. The current work is split between the paper itself, conceptual spacetime sketches, and toy-model canvases that show the difference between an unstabilized massless picture and a model with an added short-distance support term.

Paper Scope Effective-model framing first, stronger microscopic claims later.
Main Split Original massless attraction-only picture versus support-enabled finite-radius picture.
