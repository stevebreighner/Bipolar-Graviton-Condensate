What The Equation Does

The model defines V(r) = A/r^12 - B/r^3 - 1/r and computes the force from F(r) = -dV/dr = 12A/r^13 - 3B/r^4 - 1/r^2.

  • A / r12: short-range support term. If A > 0, it grows fastest at small r and can prevent collapse to r = 0.
  • -B / r3: attractive bipolar binding term.
  • -1 / r: gravity term in normalized units with G = 1 and M = 1.

The red point is the computed stable radius r*, found where F(r) = 0 and the equilibrium is stable. If A = 0 with the current attractive terms, no finite stable radius appears in this static model.

The second chart plots v(r) = sqrt(r * |F(r)|). The blue curve uses the full bipolar force, while the dashed gray curve uses the Schwarzschild/Newtonian baseline F(r) = -1/r^2. Their difference isolates the effect of the 1/r^3 binding term away from the pure gravity case.

Working Notes (From This Discussion)

  • This app is a toy model: it shows what follows if we assume V(r) = A/r^12 - B/r^3 - 1/r.
  • The stable radius r* is the center-to-center separation where net radial force is zero and locally stable.
  • The dipole term -B/r^3 is attractive; by itself (with gravity) it does not create a finite stable radius.
  • The short-range term +A/r^12 is an added regularization/support term that prevents collapse to r=0 in this model.
  • So this demonstrates model consistency, not experimental proof of a real graviton condensate.
  • Schwarzschild baseline here means gravity-only behavior; setting A=0 and B=0 recovers that baseline potential.
  • Open theory question to ponder: what physically sets the nonzero separation scale (hard core, quantum pressure, self-interaction, or another mechanism)?
  • Current interpretation: “the cloud can be stabilized if a short-distance support mechanism exists.”
