How To Use

  • Click a box to edit it.
  • Undo reverses your last tree change.
  • Width: Book uses a narrower layout for document insertion.
  • Zoom controls help with dense branches and presentation prep.
  • Shift + drag or Shift + click to select nodes, then click New Tree from Selection.
  • Focus Selected shows only the selected branch to reduce clutter.
  • Collapse/Expand toggles selected node children visibility.
  • Save Snapshot / Load Snapshot stores named versions you can restore.
  • Right-click a node to open a new tree tab from that branch.
  • Double-click a node to edit text directly in the node.
  • Alt + double-click to highlight that full path in red.
  • Save to Folder writes/overwrites one JSON file on disk.
  • Add Fork creates a branch below the selected node.
  • Add Sibling adds an alternative at the same level.
  • Delete Above removes selected node and ancestors, keeping lower branches.
  • Download JSON to keep backups.
  • Export JPG for quick insertion in docs.
  • Print / PDF opens a clean printable version.
